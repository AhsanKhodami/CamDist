## [1.0.0] - 2025-07-25

### Added
- Initial release of CamDistance
- Automatic camera distance detection using facial landmark detection
- Integration with jsPsych experimental framework
- Real-time face detection using MediaPipe Face Mesh
- Geometric distance calculation based on interpupillary distance
- Temporal smoothing and outlier rejection algorithms
- Web-based implementation requiring no software installation
- Support for modern browsers with WebRTC capabilities
- Configurable parameters for different experimental setups
- Real-time feedback and stability indicators
- Comprehensive documentation and examples
- MIT license for open-source use
